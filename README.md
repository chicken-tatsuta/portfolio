## Tatsuta Motoki Portfolio

One-page portfolio site for internship applications, built with Next.js, TypeScript, and Tailwind CSS.

The site is designed to position Hanabi experience as the core story:

- Team building and leadership
- Fundraising and outreach
- Branding, storytelling, and operations
- Startup-friendly presentation with minimal visuals

## Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Verification

- `npm run lint`
- `npm run build`
